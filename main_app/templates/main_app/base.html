What is this?

Third-party application patching is a common and potentially time-consuming issue in enterprise system administration. The Early Warning System is a free open source webcrawler that is constantly monitoring the latest versions and download urls for {{ appscount }} common enterprise applications.

You can use The Early Warning System API to help you fully automate third party application patching for SCCM/Intune. Check out our example powershell scripts in github.

You can also sign up for free automated notification emails from The Early Warning System. Choose which software updates you would like to be notified of and the system will send an automated email for you once a new version is detected.
