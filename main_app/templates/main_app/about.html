About The Early Warning System:

This tool was created to help system administrators keep track of common enterprise software updates. This automated system is constantly crawling software vendor websites for the latest version data. Once a new version of some software is detected an automated email is sent to the list of accounts who have subscribed to that specific software. Users can choose which application version changes they want to notified of. Users can change their "subscriptions" at any time.

How often does the system check for version changes?

The system checks vendor websites every 6 hours.

Get the software information via API

You can access the API from: this link.

Below you will find an example of how to access the information via python script:

import requests

url = "https://www.earlywarningsys.net/api/versions/?format=json"
response = requests.get(url)
json = response.json()

for key in json:
    if key['software'] == 'Citrix Receiver':
        print(key["version"])
