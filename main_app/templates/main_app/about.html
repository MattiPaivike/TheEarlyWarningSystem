About The Early Warning System:

This tool was created to help system administrators keep track of common enterprise software updates. This automated system is constantly crawling software vendor websites for the latest version data. Once a new version is detected an automated email is sent to the list of accounts who have subscribed to that specific software. Users can choose which application version changes they want to notified of. Users can change their subscriptions at any time. If you no longer want to receive emails from this site then just sign in and delete your account from "account settings"

How often does the system check for version changes?

The system checks vendor websites every 6 hours.
